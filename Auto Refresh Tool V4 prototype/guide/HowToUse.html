How to use the tool


Step 1 : Find Website

Use a browser to navigate to specifc subpage that you would like to monitor. Then copy the URL for the page.


Step 2 : Locate Files

You need to locate the folder on your machine called,"Auto Refresh Tool V4". This is the program folder. The folder can be found through the search functionallity in file explorer. The folder is likley in the dowloads folder.


Step 3 : Set Websites

Launch the configuration tool,found in the program folder, by opening the file called,"configuration tool".Then select the,"add a new website" button, in blue. Enter a name for the webpage, that will inform you as to what it is, into the name feild. Paste the URL of the webpage into the URL field. Then press the green, "add stite" button. This will return you to the home page of the configuration tool. You should see the name of the webpage on the list of webpages. In order to remove a webpage, navigate to the webpage secion from the home page and press the "remove this webpage" buttton twice.


Step 4 : Set Keywords

From the home page, click on the button of the webpage that you have added. From here, enter new keywords into the field above the green button. Then press the green,"Add new keyword" button. The keyword should then appear on the page in a red box. Add keywords that are than three letters or more in length in order to avoid casing false positive errors. Keywords are not case sensitve. The more relevent keywords that you add, there is a lower chance that a meaningful webpage update will me missed by the tool. Keywords can be removed by clicking on the red button with the keyword found in the are labled witht the webpage name. Return to the main menu by pressing the,"Go to home page" button.


Step 5 : add email

Click the blue,"configure emails" on the home page. Enter the email address that you would like to notify in the feild above green button. Then press the green,"Add new email button". Emails can be removed by pressing the red button with the the email found in the email configuration page.


Step 6 : launch the tool

On the home page, press the green button labeled,"start the tool". A black window will then pop up. Only whilst this window is open will the tool be activated. The tool will only notify you of changes that occur whilst this window is open. You can stop the tool by closing the window.



When a webpage changes and there is a change in the keywords on the page, the tool will email all of the listed emails. The email will be from: autoemailmachine@gmail.com and the email will have,"Refresh Tool" in the subject.
