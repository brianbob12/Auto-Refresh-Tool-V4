unexpected crash or program won't start

The most likley cause of a crash is corrupted perameters. In order to fix this open the program folder and run the resetPerameters python file. After this try to re-enter the website information and try again. If the issue persists it is likely due to incorect input data such as incorrect URL.

An unexpected crash could also occur due to mising files. To check that all files are present, open the program and confirm the presence of the following essencial files:

  • configuration tool -python file
  • curl - application
  • emails - text document
  • keywords - text document
  • main - python file
  • watchList - text documet
If one of thease files are missing, reinstall Auto Refresh Tool.

Reciving Emails Too Ofen

This issue is caused by the selected keywords changing constantly between website loads. The keyword and website at fult can be identified in the "keywords:" section of the email. Remove this keyword by navigating to the page for that webpage fromt he main menu and clicking on the red box containing this keyword.

Not Reciving Emails When Webpage Is Updated

This is most likley caused because the keywords are not applicable for this specific page or update. To fix this, more keywods should be added.
